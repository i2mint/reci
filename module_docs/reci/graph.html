reci.graph

Recipe DAG — computational graph built on graphlib.

RecipeGraph wraps graphlib.TopologicalSorter to model the two-level structure of GitHub Actions workflows: jobs containing sequential steps, with inter-job parallelism governed by needs: edges and data-flow dependencies.

Each wave is a tuple of node keys that can execute concurrently. Raises CycleError if the graph has cycles.
